🏥 Hospital Management System
📌 Overview

This project is a basic Hospital Management System developed using C along with simple web technologies (HTML, CSS, JavaScript). It demonstrates core hospital operations like managing patient records, doctor details, and basic interface design.

The project is mainly built for academic learning, focusing on programming concepts and system structure rather than full-scale deployment.

⚙️ Tech Stack
C Programming – Core logic and system functionality
HTML, CSS, JavaScript – Basic frontend interface
Platform: Windows
📂 Project Structure
hospital-management/
│── hospital.c        # Core C program
│── index.html        # Basic UI page
│── style.css         # Styling
│── script.js         # Frontend logic
│── .vscode/          # Editor configuration
│── .dist/            # Project-related files (if any)
🎯 Features
Basic patient and doctor management (C program)
Menu-driven system logic
Simple frontend interface for demonstration
Modular approach for future expansion
